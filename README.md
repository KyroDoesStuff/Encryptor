# Encryptor - A ceaser Cipher encryption tool

Encryptor is a simple terminal based encryption/deceyption tool made in C
It follows the ceaser cipher method for encrypting/decrypting text messages

# features: 
- Encryption
- Decryption
- Treats Uppercase and Lowercase differently during encryption

# how to run: 
1. Compile the program: 
    gcc main.c functions.c -o main  

2. run the executable:
    ./main.exe
